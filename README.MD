# Angular-SpringBoot-Template

Stack:
* Angular 5
* Spring Boot
* MyBatis
* Liquibase
* Gradle

# Project build:

`cd frontend`

`npm install`

`cd ..`

`gradlew clean build`

# Launching

`before launching create DB with name "test_db"`

`java -jar [path to result jar]`

`then open link in browser http://localhost:8082/#/manage`
