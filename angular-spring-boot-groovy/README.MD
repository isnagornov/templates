Angular-SpringBoot-Groovy-MyBatis-Liquibase-Template

Stack:
* Angular 5
* Spring Boot
* MyBatis
* Liquibase
* Gradle

# Project build:

`cd [projectPath]/common-angular-frontend`

`npm install`

`cd [projectPath]`

`gradle wrapper`

`gradlew clean build`

# Launching

`execute in terminal command *java -jar [path to result jar]*`

`then open link in browser http://localhost:8082/#/manage`
